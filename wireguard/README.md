# WireGuard 設定紀錄

本資料夾用來紀錄 self-sovereign-node 中 VPN 的設定流程、踩雷、客戶端設定等。

## 📡 功能概述

- 提供加密遠端連線  
- 手機 / 筆電都能安全連入家中節點  
- 支援 Pi 當 Gateway（流量經由樹莓派上網）  
- 支援多 Client、跨網路（家用網路、公網、4G 熱點）

---

## 📁 內容規劃

- `server-example.conf`：伺服器端設定  
- `client-example.conf`：客戶端設定  
- `gateway-notes.md`：VPN 流量轉送與上網 還有當初測試的一些心得

# Nextcloud 私有雲設定

本資料夾用來紀錄 self-sovereign-node 中「私有雲（Nextcloud）」的完整設定流程。

Nextcloud 在本專案中主要用途：

- 作為個人資料的存放區，讓設備資料同步，除了google、MS、Dropbox外，還有其他選擇  
- 上傳到公開資料夾後自動與IPFS串接、自動產生CID 並PIN


## 📁 內容規劃（目前進度）

- `config-example.php`：Nextcloud 設定範本
- `public-folder-notes.md`：公開資料夾與 IPFS 同步設計說明
- `install-notes.md`：安裝與升級流程（待補，先列框架）
- `issues.md`：遇到的問題與解法、效能與磁碟空間相關踩雷（待補）

# Nextcloud 私有雲設定

本資料夾用來紀錄 self-sovereign-node 中「私有雲（Nextcloud）」的完整設定流程。

Nextcloud 在本專案中主要用途：

- 作為個人資料的主存放區  
- 文件集中化管理  
- 上傳到公開資料夾後自動與IPFS串接、自動產生CID 並PIN
